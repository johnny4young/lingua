Lingua ships the everyday panels that usually force a code-runner workflow into browser tabs. They run inside the app next to the editor and result panel.

## Data

- <span id="json-formatter"></span>**JSON Formatter** — format, inspect, and repair JSON.
- <span id="yaml-json"></span>**YAML / JSON** — convert between YAML and JSON.
- <span id="json-csv"></span>**JSON / CSV** — convert tabular payloads between JSON and CSV.
- <span id="sql-formatter"></span>**SQL Formatter** — format SQL for readable diffs.
- <span id="number-base"></span>**Number Base** — convert decimal, binary, hex, and related bases.

## Text

- <span id="beautify-minify"></span>**Beautify / Minify** — reflow supported text formats.
- <span id="string-case"></span>**String Case** — convert naming conventions.
- <span id="string-inspector"></span>**String Inspector** — inspect invisible characters and string metadata.
- <span id="regex"></span>**Regex Tester** — test patterns against sample text.
- <span id="diff"></span>**Diff Viewer** — compare two text values.
- <span id="markdown-preview"></span>**Markdown Preview** — render Markdown without leaving Lingua.
- <span id="lorem-ipsum"></span>**Lorem Ipsum** — generate placeholder copy.
- <span id="random-string"></span>**Random String** — generate random tokens.

## Web

- <span id="html-entity"></span>**HTML Entity** — encode and decode HTML entities.
- <span id="url-parser"></span>**URL Parser** — break URLs into readable parts.
- <span id="url-encode"></span>**URL Encode** — encode and decode URL components.
- <span id="curl-to-code"></span>**curl to Code** — turn a curl command into code.
- <span id="html-to-jsx"></span>**HTML to JSX** — convert markup for React surfaces.
- <span id="backslash-escape"></span>**Backslash Escape** — escape and unescape text.

## Crypto

- <span id="base64"></span>**Base64** — encode and decode Base64 text.
- <span id="base64-image"></span>**Base64 Image** — inspect image data URIs.
- <span id="jwt"></span>**JWT Decoder** — decode JWT headers and payloads.
- <span id="hash"></span>**Hash** — calculate common hashes.
- <span id="uuid"></span>**UUID** — generate UUID values.

## Visual And Time

- <span id="color"></span>**Color Converter** — convert color formats.
- <span id="svg-to-css"></span>**SVG to CSS** — prepare SVG assets for CSS use.
- <span id="qr"></span>**QR Generator** — generate QR codes.
- <span id="cron"></span>**Cron Parser** — explain cron expressions.
- <span id="timestamp"></span>**Timestamp Converter** — convert timestamps and dates.
