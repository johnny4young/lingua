Lingua incluye los paneles cotidianos que normalmente empujan un flujo de code runner hacia varias pestañas del navegador. Corren dentro de la app junto al editor y el panel de resultados.

## Datos

- <span id="json-formatter"></span>**Formateador JSON** — formatea, inspecciona y repara JSON.
- <span id="yaml-json"></span>**YAML / JSON** — convierte entre YAML y JSON.
- <span id="json-csv"></span>**JSON / CSV** — convierte payloads tabulares entre JSON y CSV.
- <span id="sql-formatter"></span>**Formateador SQL** — formatea SQL para diffs legibles.
- <span id="number-base"></span>**Base numérica** — convierte decimal, binario, hex y bases relacionadas.

## Texto

- <span id="beautify-minify"></span>**Embellecer / Minificar** — refluye formatos de texto soportados.
- <span id="string-case"></span>**Case de texto** — convierte convenciones de nombres.
- <span id="string-inspector"></span>**Inspector de texto** — inspecciona caracteres invisibles y metadata de strings.
- <span id="regex"></span>**Tester Regex** — prueba patrones contra texto de ejemplo.
- <span id="diff"></span>**Visor Diff** — compara dos bloques de texto.
- <span id="markdown-preview"></span>**Preview Markdown** — renderiza Markdown de forma segura.
- <span id="lorem-ipsum"></span>**Lorem Ipsum** — genera copy placeholder.
- <span id="random-string"></span>**Texto aleatorio** — genera strings con longitud y alfabeto controlados.

## Web

- <span id="html-entity"></span>**Entidad HTML** — codifica y decodifica entidades.
- <span id="url-parser"></span>**Parser URL** — divide URLs en partes inspeccionables.
- <span id="url-encode"></span>**URL Encode** — codifica y decodifica payloads URL-safe.
- <span id="curl-to-code"></span>**curl → Código** — convierte comandos curl en snippets.
- <span id="html-to-jsx"></span>**HTML → JSX** — adapta markup HTML a JSX.
- <span id="backslash-escape"></span>**Escape backslash** — prepara strings para pegar en código.

## Cripto e identificadores

- <span id="base64"></span>**Base64** — codifica y decodifica texto.
- <span id="base64-image"></span>**Imagen Base64** — inspecciona payloads de imagen inline.
- <span id="jwt"></span>**Decodificador JWT** — lee header y payload sin enviar tokens a un servidor.
- <span id="hash"></span>**Hash** — calcula hashes comunes.
- <span id="uuid"></span>**UUID** — genera y valida UUIDs.

## Visual y tiempo

- <span id="color"></span>**Conversor de color** — convierte entre formatos de color.
- <span id="svg-to-css"></span>**SVG → CSS** — prepara SVGs para uso en CSS.
- <span id="qr"></span>**Generador QR** — crea QR codes localmente.
- <span id="cron"></span>**Parser Cron** — explica expresiones cron.
- <span id="timestamp"></span>**Timestamp** — convierte fechas y timestamps.
