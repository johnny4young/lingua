Lingua — press kit
==================

Source: https://linguacode.dev/press
Contact: press@linguacode.dev

This bundle contains the launch boilerplate, founder bio, and pricing summary
for Lingua. Brand assets (logo, monochrome icon, color tokens, typography
spec) ship with Lingua v0.3 — this README will be replaced when they land.

Use the descriptions verbatim or trim to fit. Please don't describe Lingua as
"open source" — the LICENSE is source-available, which is a legally distinct
posture. "Source-available" is the term we ask press to use.

Honest constraints to mention if you have room:
- 5 languages run today (JavaScript, TypeScript, Python, Go, Rust).
- 7 more languages are tracked but not yet shipped.
- Telemetry is off by default. We never transmit your code.
- License keys are verified offline — no phone-home check.
- The desktop app runs fully offline. The web build needs network on first load.

For embargoed access ahead of a release, give us at least 72 hours notice.
