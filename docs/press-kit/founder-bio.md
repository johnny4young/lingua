# Founder bio

Two lengths, ready to drop into articles. Keep these truthful — no
invented accolades or fake ex-FAANG credentials. Update when the
actual bio changes.

## English

### 40 words

I'm Johnny Young, the solo developer behind Lingua. I built the app
after spending too many afternoons juggling browser tabs for Go,
TypeScript, Python, and Rust snippets. Lingua pulls them into one
Monaco-powered desktop app.

### 100 words

I'm Johnny Young, the solo developer behind Lingua. I started building
multi-language tooling because I wanted one desktop app that could run
a Go snippet, a Python function, a TypeScript proof-of-concept, and a
Rust benchmark without opening five different web playgrounds. Lingua
is the result: a Monaco-powered code runner that treats each language
as a first-class citizen, ships built-in developer utilities, and
stays offline-first so it works on a plane. I'm releasing it as a
commercial source-available product — paid tiers unlock via
offline-verified license keys, education access is free.
