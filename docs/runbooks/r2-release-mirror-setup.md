# Retired R2 release mirror

The release-binary mirror described by older changelog entries was retired
after the repository became public. Desktop installers, checksums, compliance
artifacts, and updater manifests now come directly from GitHub Releases.

R2 remains active only for oversized web runtimes. Use the current runbook:
[`r2-web-runtime-setup.md`](./r2-web-runtime-setup.md).

This compatibility page remains so historical release notes do not contain a
dead documentation link. Do not restore `mirror-r2` or upload release binaries
to the runtime bucket.
